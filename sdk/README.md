# AMP Software Development Kits

## Overview
This directory contains the SDKs for integrating the AMP protocol into various game engines and platforms.

## Available SDKs
- **[JavaScript/TypeScript SDK](./js)**: Optimized for Web browsers and Node.js environments. Uses ethers.js and WebSockets.

## Roadmap
- **Rust SDK**: For native high-performance games.
- **Unity (C#) SDK**: For integration with the Unity game engine.
- **Unreal (C++) SDK**: For integration with the Unreal Engine.
