# AMP Trace Viewer

This is a single-page HTML application designed to load and visualize the JSON telemetry traces generated by `amp-telemetry`.

## How to Use

1. **Open the App**: Since it is a static, asset-less HTML file, you can simply open `index.html` in any modern web browser.
   ```bash
   open index.html
   # or on Linux:
   xdg-open index.html
   # or manually double-click it.
   ```

2. **Generate a Trace**:
   Ensure you have ran `./test_mvp.sh` or executed one of the localized SDK tests while `amp-telemetry` was running. The telemetry receiver will output JSON payloads into the `amp-telemetry/` directory or its stdout depending on configuration. Capture or save a valid telemetry JSON array to a file.

3. **Load**:
   Click the **"Load Trace JSON"** button in the top right and select your JSON file.

4. **Analyze**:
   Click on any span block in the timeline to view its detailed execution attributes, timing, and source service in the details pane.
